## ✅ Issue 2: 사업공고 파일 첨부 미구현 & 20MB 용량 제한 (Pages 2-4) - **완료**

**위치:** 홈 > 사업공고 > 신청기록 > 추가 자료 제출 모달  
**현상:**

1. 파일 첨부(추가 자료 제출) 기능 자체가 미구현 (Page 4: "기능 준비 중입니다" 알림)
2. 용량체크는 작동하나 파일 첨부 자체가 안 됨 (Page 2: "용량체크는 작동 > 첨부 안됨")
3. 첨부 파일 용량 제한이 10MB로 표기되어 있으나 20MB로 변경 필요 (Page 3)
   **우선순위:** 높음  
   **영향 범위:**

- `frontend/src/member/modules/projects/` (신청기록, 추가 자료 제출 모달)
- `backend/` (파일 업로드 API, 용량 검증)

### 수정 방향

- 추가 자료 제출 파일 업로드 API 구현/연동
- 파일 크기 제한을 10MB → 20MB로 변경 (프론트엔드 & 백엔드)
- 파일 크기 초과 시 적절한 에러 메시지 표시
- 지원 형식: PDF, DOC, XLS

## ✅ Issue 13: 지원사업 보완요청 기능 추가 (Page 16) - **완료**

**위치:** 관리자 > 지원사업 신청 목록
**현상:** 심사중 상태의 신청에 "보완 요청" 버튼 추가 필요
**우선순위:** 높음
**영향 범위:**

- `frontend/src/admin/modules/projects/` (신청 관리 페이지)
- `backend/` (보완 요청 API, 상태 관리)
- `frontend/src/member/modules/projects/` (보완 요청 수신 화면)

### 수정 방향

- 관리자: 심사중 상태에서 보기|기업|승인|거절 옆에 "보완 요청" 버튼 추가 ✅
- 보완 요청 시 사유 입력 모달 ✅
- 사용자: 보완 요청 알림/상태 표시, 추가 자료 제출 가능 ✅

### 상태 전이 (State Machine)

```
심사중 ──(관리자: 보완요청)──→ 보완요청
보완요청 ──(사용자: 자료보완 제출)──→ 서류보완완료
서류보완완료 ──(관리자: 재심사)──→ 심사중
심사중 ──(관리자: 승인)──→ 승인
심사중 ──(관리자: 거절)──→ 거절
```

- 백엔드에서 상태 전이 규칙 검증 필수 ✅
- 보완요청 상태의 신청은 사용자가 추가 자료 제출 UI 접근 가능 ✅
- 서류보완완료 → 심사중 전환은 관리자가 수동 확인 후 처리 ✅
