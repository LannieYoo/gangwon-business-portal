# Build and Fix

逐步修复构建错误：

1. 运行构建: `npm run build` 或 `npm run dev`

2. 解析错误输出：
   - 按文件分组
   - 按严重性排序

3. 对每个错误：
   - 显示错误上下文（前后 5 行）
   - 解释问题
   - 提议修复
   - 应用修复
   - 重新运行构建
   - 验证错误已解决

4. 停止条件：
   - 修复引入新错误
   - 同一错误在 3 次尝试后仍然存在
   - 用户请求暂停

5. 显示摘要：
   - 已修复的错误
   - 剩余错误
   - 引入的新错误

一次修复一个错误以确保安全！

## 项目特定错误类型

**Vite 构建错误:**
- 模块解析错误
- 路径别名问题
- 环境变量错误（使用 `import.meta.env`）

**React/JSX 错误:**
- 缺少导入
- Hook 规则违反
- 组件渲染错误

**i18n 错误:**
- 翻译键不存在
- useTranslation 未导入

**Zustand 错误:**
- create 未导入
- Store 类型错误
