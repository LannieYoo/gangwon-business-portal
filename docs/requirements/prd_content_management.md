## 3. 功能详情

### 3.1 公告管理

**创建公告**:

- 标题 (必填)
- 内容 (WYSIWYG富文本编辑器)
- 附件上传 (最多3个文件)
- 内嵌图片

**列表功能**:

- 搜索
- 分页
- 排序 (创建时间)

**API接口**:

- `GET /api/v1/notices/latest5` - 首页最新5条

### 3.2 新闻管理

**创建新闻**:

- 标题 (必填)
- 图片上传 (必填)
- 自动生成缩略图

**列表功能**:

- 预览图显示
- 分页

**API接口**:

- `GET /api/v1/press/latest1` - 首页最新1条

### 3.3 Banner管理

**Banner类型**:
| 类型 | 位置 | 说明 |
|------|------|------|
| MAIN | 首页主Banner | 全宽轮播 |
| INTRO | 系统介绍页 | 顶部Banner |
| PROGRAM | 项目页 | 顶部Banner |
| PERFORMANCE | 绩效页 | 顶部Banner |
| SUPPORT | 支持页 | 顶部Banner |

**配置项**:

- 图片上传
- 链接URL
- 激活状态 (开启/关闭)
- 轮播顺序

### 3.4 FAQ管理

**创建FAQ**:

- 问题 (必填)
- 答案 (必填)
- 分类
- 显示顺序

**前端展示**:

- 手风琴式展开
- 分类筛选
- 搜索功能

### 3.5 系统介绍管理

**编辑内容**:

- WYSIWYG富文本编辑器
- 单图上传
- 仅支持更新 (单条记录)

### 3.6 UI/UX 体验

**富文本编辑器**:

- 标题格式
- 粗体/斜体/下划线
- 列表 (有序/无序)
- 图片插入
- 链接添加

**文件上传**:

- 拖拽上传
- 进度显示
- 类型和大小验证

## 5. 技术注解

### API 端点

| 方法     | 端点                        | 描述        |
| -------- | --------------------------- | ----------- |
| GET/POST | `/api/v1/admin/notices`     | 公告CRUD    |
| GET      | `/api/v1/notices/latest5`   | 最新5条公告 |
| GET/POST | `/api/v1/admin/press`       | 新闻CRUD    |
| GET      | `/api/v1/press/latest1`     | 最新1条新闻 |
| GET/POST | `/api/v1/admin/banners`     | Banner管理  |
| GET/POST | `/api/v1/admin/faqs`        | FAQ管理     |
| GET/PUT  | `/api/v1/admin/system-info` | 系统介绍    |

### 文件存储结构

```
/upload
  /{businessId}
    /notice          # 公告附件和内嵌图片
    /press           # 新闻图片
  /common
    /system-info     # 系统介绍图片
    /banners         # Banner图片
```
