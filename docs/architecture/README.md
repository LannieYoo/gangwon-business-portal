## 架构概览

```
┌─────────────────┐      ┌─────────────────┐
│  Member Portal  │      │   Admin Portal   │
│   (React SPA)   │      │   (React SPA)    │
└────────┬────────┘      └────────┬────────┘
         │                        │
         └──────────┬─────────────┘
                    │ HTTPS/REST
         ┌──────────▼──────────┐
         │   FastAPI Backend   │
         │   (API Gateway)     │
         └──────────┬──────────┘
                    │
    ┌───────────────┼───────────────┐
    │               │               │
┌───▼───┐    ┌──────▼──────┐  ┌─────▼─────┐
│PostgreSQL│  │   Storage   │  │Nice D&B API│
│(Supabase)│  │ (Supabase)  │  │  (External)│
└─────────┘  └─────────────┘  └───────────┘
```
